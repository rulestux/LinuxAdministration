### Decimal to Binary Conversion

- dividir sucessivamente o valor por 2, até que o quociente seja 1; 105, por exemplo:

 ```
105/2
Quociente = 52
Resto = 1

52/2
Resto = 0
Quociente = 26

26/2
Resto = 0
Quociente = 13

13/2
Resto = 1
Quociente = 6

6/2
Resto = 0
Quociente = 3

3/2
Resto = 1
Quociente = 1
 ```

- agrupar o último quociente seguido pelo resto de todas as divisões da direita para a esquerda:

 ``` 1101001 ```

- incluir zeros à esquerda até atingir 8 bits:

 ``` 01101001 ``` = 105 binário.

### Binary to Decimal Conversion

- associar cada bit a uma potência na base 2, iniciando com zero e incrementando da direita para a esquerda; 
- quando o bit for 1, atribui-se a potência respectiva; quando for 0, atribui-se 0;
- binário 10110000, por exemplo:

| 1		| 0		| 1		| 1		| 0		| 0		| 0		| 0		|
|-------|-------|-------|-------|-------|-------|-------|-------|
| 2⁷	| 2⁶	| 2⁵ 	| 2⁴	| 2³ 	| 2²	| 2¹	| 2⁰ 	|
| 128	| 0		| 32 	| 16	| 0		| 0		| 0		| 0 	|

- somam-se os valores: 128 + 32 + 16 = 176, que é o decimal equivalente a 10110000.

### Netmask
- máscara de rede que determina qual parte do IP representa a rede e qual representa os hosts:

| Decimal			| Binário								| CIDR	|
|-------------------|---------------------------------------|-------|
| 255.0.0.0			| 11111111.00000000.00000000.00000000	| /8	|
| 255.255.0.0		| 11111111.11111111.00000000.00000000	| /16	|
| 255.255.255.0		| 11111111.11111111.11111111.00000000	| /24	|

- 255 define rede/sub-rede | 0 define hosts;
- CIDR \(Classless Inter-Domain Routing) é uma notação de máscara simplificada, que indica o número de bits associado à rede/sub-rede;

### Network Ports
- portas são identificadas por campos de 16 bits, resultando num limite de 65.535 portas, das quais de 1 a 1023 são privilegiadas e de 1024 a 65535, não privilegiadas ou socket;
- portas padronizadas conforme Internet Assigned Numbers Authority - IANA:
	- 20 FTP dados
	- 21 FTP controle
	- 22 SSH
	- 23 Telnet acesso remoto
	- 25 SMTP receber email
	- 53 DNS
	- 80 HTTP
	- 110 POP3 receber email
	- 123 NTP
	- 139 Netbios
	- 143 IMAP acessar email
	- 161 SNMP monitoramento de informações
	- 162 SNMPTRAP notificações SNMP
	- 389 LDAP
	- 443 HTTPS
	- 465 SMTPS
	- 514 RSH
	- 636 LDAPS
	- 993 IMPAS
	- 995 POP3S
- as portas são listadas no arquivo '__/etc/services__'.

### Protocols
- ICMP Internet Control Message Protocol: Echo Request - Echo Reply: testes Ping e Traceroute;
- TCP Transmission Control Protocol: protocolo seguro de transporte, com controle de retransmissão de segmentos;
- UDP User Datagram Protocol: protocolo ágil, simples, mas sem controle;
